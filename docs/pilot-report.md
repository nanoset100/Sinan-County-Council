# §12-3 A1 품질 파일럿 실측 리포트

> 본 가동 전 필수 파일럿(PRD §12-3). **모델·프롬프트·Verifier·비용을 실측으로 확정**한 기록.

## 조건
- **모델:** `claude-sonnet-5` (temperature 미사용 — 이 모델군은 샘플링 파라미터를 400 으로 거부).
- **표본:** 신안군의회 **제334회 제1차 행정복지위원회(2026-03-09, 제9대)** 실안건 **조례안 12건**
  (제정 5·전부개정 1·일부개정 5·폐지 1). 근거 원문 = 각 안건 제안설명.
  ※ 이 표본은 §5 게시 범위(제10대~) 밖이므로 **품질 측정 자산으로만 보관**하고 게시하지 않는다.
- **실행:** `npm run pilot -- --input=fixtures/pilot/mine.json --n=12` (`.env` 의 키·모델 자동 로드).

## 결과

| 지표 | 1차(초안 프롬프트) | 2차(**확정 프롬프트**) |
|---|---|---|
| Verifier 통과율 | 91.7% (11/12) | **100% (12/12)** |
| 오탐(false positive) | 0 | 0 |
| 건당 생성시간 | 7.45초 | 6.32초 |
| 토큰(입력/출력) | 12,580 / 8,903 | 15,448 / 7,573 |
| 총 비용(추정) | $0.171 | **$0.160** |
| **건당 비용** | — | **약 $0.013** |

- **비용 실측:** 건당 약 **$0.013**. 정례회 1회기 안건 40건 기준 **약 $0.5**, 100건이라도 **약 $1.3**.
  → PRD 의 "회기당 수 달러" 추정은 **실측상 그보다 낮음**(수십 센트~1달러대).

## 품질 소견 (사람 확인)
- **수사 제거 확인:** 출생기본소득안(제2341호)의 원문 수사("인구소멸이라는 거대한 파도", "강력한
  의지", "아이들의 웃음소리가 사라지는 섬")를 **전부 걷어내고 사실(제2·4·5·12조·관계법령·입법예고)만
  재서술**. 원칙 7·§14-1 준수.
- **환각 없음:** 시행일이 원문에 없으면 "명시되어 있지 않음"으로 정직 표기. 지어내지 않음.
- **1차의 유일 실패 = 정당한 차단:** 제2341호의 앵커 인용문이 "..." 로 서로 떨어진 문장을 이어붙여
  원문의 연속 구간이 아니었음 → Verifier 가 정상 차단(요약 내용 자체는 정확). **오탐 아님.**

## 확정 프롬프트 변경 (1차 → 2차)
`A1_SYSTEM_PROMPT` 에 **앵커 quote 규칙**을 추가:
> quote 는 원문의 연속된 한 구간을 글자 그대로 복사한다. 축약·요약·바꿔쓰기 금지, 떨어진 문장을
> "..."로 이어붙이지 않는다. Verifier 는 그대로(엄격) 유지.

이 한 줄로 통과율 91.7% → 100%. Verifier 는 변경하지 않음.

## Verifier 한계와 검수 (남은 §12-3 항목)
- Verifier 통과는 정확성 증명이 아니다. **절차 동사(원안가결↔부결, 제정↔개정↔폐지 등) 혼동**과
  **방향 오류(인상/인하, 증원/감축)** 는 자동 검증이 못 잡는다 → **사람 검수가 유일한 게이트**.
  체크리스트: [review-checklist.md](./review-checklist.md).
- 건별 결과 + `human_label`(오탐/미탐 판정란)은 `.pipeline-out/pilot/*.json` 에 있다.
- **건당 실제 검수 소요 시간**(§6-1 산정표 교체용)은 운영자가 실제 검수하며 측정한다.

## 결론
모델 `claude-sonnet-5` + 확정 프롬프트 + 현행 Verifier 조합으로 **A1 본 가동 가능**. 사이트 공개 전
남은 조건은 코드가 아니라 사람 작업(§15 법률 검토, §12-4 검수자 확보)이다.
