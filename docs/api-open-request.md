# 신안군 Open API 개방·데이터 반영 요청 (초안)

§12-G0 판정 후속(PRD v0.5). 국회도서관 지방의정포털에 **신안군의회 제10대 데이터 반영 및 Open API
rasmblyId 개방**을 요청하는 메일/오류신고 초안이다. 발송은 사람이 검토 후 진행한다.

## 보내는 곳
- **메일:** `lawstat@nanet.go.kr` (Open API 담당, 데이터융합분석과)
- **전화 문의(병행):** 02-6788-4475 / 리소스센터 02-6788-4150
- **오류신고:** 포털 하단 `오류신고` 메뉴로도 동일 내용 접수 가능

## 메일 초안 (복사용)

```
받는사람: lawstat@nanet.go.kr
제목: [데이터 반영 요청] 지방의정포털 신안군의회 의안·회의록(제10대) 및 Open API rasmblyId 개방 요청

안녕하세요. 국회도서관 지방의정포털 Open API 를 이용해 신안군 의정활동을 공개된 공식
기록에 근거해 비영리·중립적으로 재공개하는 시민 정보 서비스를 준비 중인 이용자입니다.

Open API(회의록/의안정보/의원정보) 인증키를 신청(계정: 장경수)했고, 신안군의회 데이터를
연계하려는 과정에서 아래 사항을 확인해 문의드립니다.

1) 의안정보(bill.do): rasmblyId '목록 보기'에 신안군의회가 없고, 웹 상세검색에서도
   신안군 의안이 전 기간·전 대수 0건으로 조회됩니다.
2) 회의록(minutes.do): 웹 검색상 신안군 회의록은 제6대(2013년)까지만 확인되고,
   제10대(2026-07 개원) 등 최근 회기는 반영되어 있지 않습니다.

이에 다음을 요청드립니다.
 (가) 신안군의회 의안·회의록의 최신 회기(특히 제10대, 2026-07~) 데이터 반영 일정 안내
 (나) 신안군의회의 Open API rasmblyId 개방(의안·회의록·의원정보) 가능 여부 및 절차
 (다) 개방이 신안군의회의 동의를 요하는 경우, 필요한 협조 절차 안내

서비스는 특정 정당·후보와 무관한 비영리 시민 정보 제공이며, 모든 콘텐츠에 원문 출처를
링크합니다. 회신 주시면 감사하겠습니다. 감사합니다.

(성명 / 연락처 / 이메일)
```

## 신안군의회에도 병행 요청(선택)
포털 개방이 신안군의회 홈페이지 게시·동의에 달려 있을 수 있으므로, 신안군의회 사무국에도
"의안·회의록의 지방의정포털 연계/공개 협조"를 함께 문의하면 처리가 빨라질 수 있다.

## 참고 — 요청이 받아들여지면
`COLLECTOR=s1` 로 전환하고 `npm run g0` 로 신안군 rasmblyId 반환을 확인한 뒤, S1Collector 의
필드 매핑(이미 실제 명세로 구현됨)을 최종 점검한다. 상세: [G0-api-key-request.md](./G0-api-key-request.md).
